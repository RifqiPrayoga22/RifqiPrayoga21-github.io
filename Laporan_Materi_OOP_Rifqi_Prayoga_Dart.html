1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Objek dibuat dari sebuah kelas (class) yang berisi atribut (data) dan method (fungsi).

2. Contoh Class dan Object


class Car {
  String model;

  Car(this.model);

  void start() {
    print('Car $model started!');
  }
}

void main() {
  var car = Car('Toyota');
  car.start();
}

3. Constructor & Validasi


class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk) {
    if (ipk < 0.0 || ipk > 4.0) {
      throw ArgumentError('IPK harus 0.0 - 4.0');
    }
  }
}

4. Encapsulation


class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;
  }
}

Kesimpulan

OOP mempermudah pengelolaan program besar karena kode lebih terstruktur, mudah dikembangkan, dan lebih aman melalui konsep enkapsulasi.
